关于我们/About Us

合肥工业大学HackerEmpire深度学习实验室，主要应用框架为Pytorch、MindSpore。目前已开发项目有“EyeNet：基于PyTorch的青光眼眼底病变识别辅助模型”等。

HackerEmpire Deep Learning Laboratory of Hefei University of Technology, the main application frameworks are Pytorch and MindSpore, and the projects we have developed so far include "EyeNet: Glaucoma Fundus Lesion Recognition Auxiliary Model Based on PyTorch" and so on.

计算机视觉（Computer Vision）

使用PyTorch、MindSpore深度学习框架开发性能卓越的计算机视觉感知模型，包括图像复原、目标检测、语义分割等模型开发。

Develop high-performance computer vision perceptual models using PyTorch, MindSpore deep learning frameworks, including image restoration, target detection, semantic segmentation, and other model development.

基于机器学习的数据分析（Data Analyzing Based on Machine Learning）

利用多层感知机（MLP）、支持向量机（SVM）等经典机器学习算法进行材料性能、生理指标等相关数据分析与预测模型开发。

Data analysis and predictive model development related to material properties, physiological metrics, and other relevant data using classical machine learning algorithms such as multilayer perceptual machines (MLPs) and support vector machines (SVMs).
